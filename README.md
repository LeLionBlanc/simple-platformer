# Simple Platformer

This project is a basic platformer game built with the Godot Engine. It demonstrates core platformer mechanics such as player movement, jumping, and world interaction.

## Project Structure

- `player.gd`, `player.tscn`, `player.png.import`: Player character script, scene, and sprite.
- `world.gd`, `world.tscn`: World logic and scene.
- `project.godot`: Godot project configuration file.
- `icon.svg.import`, `03 - Basilv2 - GBStudio.png.import`: Additional assets and icons.

## Getting Started

1. **Install Godot Engine**
   - Download from [godotengine.org](https://godotengine.org/).
2. **Open the Project**
   - Launch Godot and select the `simple-platformer` folder as the project directory.
3. **Run the Game**
   - Click the play button in Godot to run the main scene.

## Features

- Player movement and jumping
- Basic world/level setup
- Simple asset management

## How to Use

- Edit `player.gd` to change player behavior.
- Modify `world.gd` and related scenes for new levels or mechanics.
- Add or replace assets in the project folder.
